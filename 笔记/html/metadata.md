- author
- description
- generator
- keywords

## Other metadata names

## Pragma directives
When the http-equiv attribute is specified on a meta element, the element is a pragma directive.
- content-language
- content-type
- default-style
- refresh
- set-cookie

## Other pragma directives
https://wiki.whatwg.org/wiki/PragmaExtensions
